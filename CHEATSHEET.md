# CHEATSHEET

## vim

| SHORTCUT | ACTION |
| --- | --- |
| C-n | Toggle NERDTree |
| C-t | Toggle Tagbar |
| S-j | Vertical Pane |
| S-k | Horizontal Pane |
| C-<h-j-k-l> | Switch Pane |


## tmux

| SHORTCUT | ACTION |
| --- | --- |
| C-B + % | Vertical Pane |
| C-B + " | Horizontal Pane |
| C-B + D | Safe dettach |
| unsetmux | unset $TMUX |
| resetmux | reset $TMUX |
| ta <sn> | Attach session |
| tkss <sn> | Kill session |
